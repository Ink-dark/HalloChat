## 项目概述
即时通讯软件，包含以下功能模块：
1. 基础单聊功能（文字/语音/图片）
2. 加密聊天（端到端加密+阅后即焚）
3. 多人群组（Telegram风格）
4. 频道功能（类似Telegram/QQ频道）

## 技术栈  
- 前端：React + Electron

- 数据库：SQLite（嵌入式方案）
